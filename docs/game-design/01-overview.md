## Core Vision

**Genre:** Incremental RPG + Party Builder + Light Gacha

**Core Fantasy:**  
Grow infinitely stronger while building a team of unique companions and adapting to ever-changing challenges.

## Design Pillars

- Numbers always go up (incremental satisfaction)
- Adaptation over balance (no nerfs, only counter-content)
- Build diversity (gear + companions + class synergy)
- Character-driven progression (stories + relationships)
