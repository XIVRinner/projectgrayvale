## Crafting System

### Professions

- Blacksmithing
- Alchemy
- Expandable

### Item Sources

- Crafted gear (consistent, scalable)
- Raid gear (specialized)
- Artifact gear (unique effects)

**Design goal:**  
Crafting is a core progression and economy system.

## Economy and Trading

- NPC trading
- Player trading (future)
- 4 levels of currency (base money):
    - Penny
    - Mark
    - Crown
    - Throne
- Maybe other currencies for special activities (e.g. event currency, raid tokens) but keep it simple for now

### Currency Uses

- Gear
- Materials
- Companion interactions
- Construction (future)
- Resting and services that can only be bought with currency (e.g. inns, fast travel)

## Pocket Dimension (Base System)

### Features

- Build structures
- Assign companions
- Gain buffs

### Example Buffs

- Faster ticks
- Better crafting quality
- Passive XP gain
