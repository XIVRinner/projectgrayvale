# Gameplay Loops

[Back to index](../game-design.md) | [Previous: Overview](./01-overview.md) | [Next: Activities and Progression](./03-activities-and-progression.md)

## Primary Loop

1. Choose activity (combat / gathering / crafting / quest).
2. Activity runs on the tick system.
3. Gain:
   - XP (player + skills)
   - Loot (gear/materials)
   - Companions (events / quests)
4. Improve:
   - Equipment
   - Skills (passives)
   - Party composition
5. Unlock harder content.
6. Repeat.

## Secondary Loop (Meta Progression)

- Prestige (NG+)
- Keep:
  - Select items
  - Companions (possibly limited)
  - Permanent boosts
- Reset:
  - Level
  - World state

## Strategic Loop

- Build party with limited slots.
- Choose activities based on:
  - Efficiency
  - Build synergy
  - Current bottlenecks
- Adapt to content that counters your build.
